# credit-risk-portfolio-analysis
End-to-end Credit Risk Analysis using SQL, Python, and Power BI

📊 Credit Risk Portfolio Analysis
🔹 Project Overview

This project analyzes a credit loan portfolio to identify default risk, exposure distribution, and high-risk segments using SQL, Python, and Power BI.

🔹 Tools Used

SQL (MySQL)

Python (Pandas, Matplotlib)

Power BI

GitHub

🔹 Key Business Questions Answered

What is the overall default rate?

Which risk category has highest default probability?

How much exposure is concentrated in high-risk segments?

Which loan purposes contribute most to default exposure?

What is the portfolio distribution by risk category?

🔹 Key Insights

Overall Default Rate: 18.08%

High Risk segment shows highest default probability.

Low Risk segment holds majority of portfolio exposure.

Car & Personal loans show higher default exposure.
